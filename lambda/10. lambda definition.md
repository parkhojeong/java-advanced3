* lamdba

* terms
- lambda : anonymous function
- lambda expression: detail grammar expression () -> {}

* deal lambda like variable
```java
public static void main(String[] args) {
    Procedure procedure = new Procedure() {
        @Override
        public void run() {
            System.out.println("hello lambda");
        }
    };

    // class lambda.lambda1.InstanceMain$1
    System.out.println(procedure.getClass());
    // lambda.lambda1.InstanceMain$1@6f496d9f
    System.out.println(procedure);

    Procedure procedure2 = () -> System.out.println("hello lambda2");
    // lambda class
    //class lambda.lambda1.InstanceMain$$Lambda/0x0000000801003948
    System.out.println( procedure2.getClass());
    // lambda reference
    // lambda.lambda1.InstanceMain$$Lambda/0x0000000801003948@10f87f48
    System.out.println(procedure2);
}
```

* summary
- simple and concise
- replace anonymous class to lambda
- lambda instance creation without new keyword 
- think like syntactic sugar
